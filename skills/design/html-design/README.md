# html-design

Canonical copy lives in this repo:

`skills/design/html-design/`

## Install (symlink)

```bash
# Claude Code
ln -sfn "$PWD/skills/design/html-design" ~/.claude/skills/html-design

# Grok Build
ln -sfn "$PWD/skills/design/html-design" ~/.grok/skills/html-design

# Shared agents path (optional)
ln -sfn "$PWD/skills/design/html-design" ~/.agents/skills/html-design
```

Run from the `agent-skills` repo root. Directory name must match `name:` in `SKILL.md`.
